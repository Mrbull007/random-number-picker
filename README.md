# 🎲 Random Picker – 1 to 100,000

**A beautiful, lightweight, and fully functional random number generator**  
Built as a single-file web app — perfect for lotteries, crypto seed phrases, giveaways, games, or just pure fun.

## ✨ Features

- ⚡ Instant random number between **1 and 100,000**
- 📋 One-click copy to clipboard
- 📜 History of your last 10 picks (click any to reuse)
- ⌨️ Keyboard shortcuts (`R` or `Space`)
- 🎨 Modern, dark, mobile-friendly design (Tailwind CSS)
- 🔒 100% client-side — no backend, no tracking, no data collection
- 🚀 Zero dependencies except Tailwind CDN
- 📱 Fully responsive

## 🚀 Live Demo

After you push the code and enable GitHub Pages, your live link will be:  
`https://Mrbull007.github.io/random-number-picker`

## 🛠️ How to Run Locally

1. Download or clone this repository
2. Open `index.html` in any browser
3. Click **"GENERATE RANDOM NUMBER"** or press `R` / `Space`

No installation needed!

## 📥 Installation (GitHub Pages)

1. Go to **Settings** → **Pages**
2. Under "Source", select **Deploy from a branch**
3. Choose `main` branch → `/ (root)`
4. Click **Save**
5. Wait ~1 minute → Your app will be live!

## 🧪 Tech Stack

- HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript (no frameworks)
- Pure client-side randomness (`Math.random()`)
